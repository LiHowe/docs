## 线段 - Line
<br/>

### 使用

<Image src="https://s2.loli.net/2022/11/30/1R473TblgaAhyeZ.png" title="Line" />

```ts
// (0, 0) -> (100, 100) 的线段
new fabric.Line([
  0, 0, 100, 100,
], {
  stroke: 'rgb(0,0,0)'
})
```

:::warning 注意

构建线段对象的时候需要指定线段颜色(`stroke`默认为`null`), 否则创建的为透明线段

:::

### 图形属性及方法

绘制一条线段需要两个端点, 所以线段的属性为两个端点的x, y坐标

| 属性  | 描述     | 默认值 |
|-----|--------|-----|
| x1  | 起点的x坐标 | `0` |
| y1  | 起点的y坐标 | `0` |
| x2  | 终点的x坐标 | `0` |
| y2  | 终点的y坐标 | `0` |
