## 矩形 - Rect
<br/>

矩形为最基础的图形, 在通用对象的基础上只额外提供了边框圆角的配置.

### 使用

```ts
new fabric.Rect({
  width: 100,
  height: 40,
})
```

### 图形属性及方法

| 属性  | 描述       | 默认值 |
|-----|----------|-----|
| rx  | 水平方向边框圆角 | `0` |
| ry  | 垂直方向边框圆角 | `0` |

其中 `rx` 与 `ry` 如果只设置一个, 则表示这两个属性的值相同.
